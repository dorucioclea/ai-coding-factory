# TOB Scripts

A collection of Trail of Bits utility scripts for security testing and analysis workflows.

## Available Scripts

| Script | Purpose |
|--------|---------|
| `burp-search.sh` | Search through Burp Suite project files |

## When to Use

Use these scripts when:
- Working with Burp Suite project data
- Automating security testing workflows
- Processing security tool outputs

## Usage

Scripts can be invoked directly:

```bash
./burp-search.sh <search-term>
```

## Framework

Utility scripts supporting Trail of Bits security testing methodologies.
