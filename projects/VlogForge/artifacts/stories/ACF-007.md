# ACF-007: Team Creation and Management

**Status**: Ready
**Priority**: P1 - Must Have (MVP)
**Epic**: Teams
**Estimate**: M

## User Story

As a **creator**, I want to **create a team** so that **I can work with my editors, managers, and other team members**.

## Acceptance Criteria

### AC1: Create Team
- **Given** a creator with appropriate subscription tier
- **When** they create a new team
- **Then** a team is created with them as owner

### AC2: Invite Members
- **Given** a team owner
- **When** they invite members via email
- **Then** invitations are sent and pending invites are tracked

### AC3: Accept Invitation
- **Given** a user with pending team invitation
- **When** they accept the invitation
- **Then** they are added to the team with the specified role

### AC4: Role Assignment
- **Given** a team owner/admin
- **When** they assign roles to members
- **Then** members have roles: Owner, Admin, Editor, Viewer

### AC5: Remove Member
- **Given** a team owner/admin
- **When** they remove a team member
- **Then** the member loses access to team resources immediately

## Role Permissions

| Permission | Owner | Admin | Editor | Viewer |
|------------|-------|-------|--------|--------|
| Manage team settings | ✓ | ✓ | | |
| Invite/remove members | ✓ | ✓ | | |
| Assign tasks | ✓ | ✓ | ✓ | |
| Edit content | ✓ | ✓ | ✓ | |
| View content | ✓ | ✓ | ✓ | ✓ |

- Implement team-scoped authorization policies
