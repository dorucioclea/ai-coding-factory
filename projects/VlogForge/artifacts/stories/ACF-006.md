# ACF-006: Content Calendar

**Status**: Ready
**Priority**: P1 - Must Have (MVP)
**Epic**: Content Planning
**Estimate**: M

## User Story

As a **creator**, I want a **content calendar view** so that **I can plan and visualize my publishing schedule**.

## Acceptance Criteria

### AC1: Calendar View
- **Given** a creator with scheduled content
- **When** they view the calendar
- **Then** they see a month view with content items on their scheduled dates

### AC2: Drag and Drop Scheduling
- **Given** a creator viewing the calendar
- **When** they drag a content item to a new date
- **Then** the scheduled date updates immediately

### AC3: Quick Add from Calendar
- **Given** a creator viewing the calendar
- **When** they click on an empty date
- **Then** a quick-add modal opens pre-filled with that date

### AC4: Status Color Coding
- **Given** content items on the calendar
- **When** displayed on calendar
- **Then** items are color-coded by status (Idea=gray, Draft=yellow, Review=orange, Scheduled=blue, Published=green)

### AC5: Week/Month Toggle
- **Given** a creator viewing the calendar
- **When** they toggle view mode
- **Then** they can switch between week and month views

## Technical Notes

- Use React DnD for drag-and-drop
- Optimistic UI updates with rollback on failure
- Cache calendar data per month
